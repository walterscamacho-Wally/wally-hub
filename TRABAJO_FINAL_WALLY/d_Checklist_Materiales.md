# Checklist de Materiales e Información para Proyectos Web

Para que un proyecto como "Baila Con Wally" sea exitoso, es fundamental contar con este material de antemano:

## 1. Información de Marca (El "Qué")
- [ ] **Nombre oficial**: Exactamente como debe aparecer.
- [ ] **Propuesta de Valor**: ¿Qué te hace único? (Ej: "20 años de experiencia", "Transformación emocional").
- [ ] **Servicios Detallados**: Título, descripción corta y beneficio de cada uno.
- [ ] **Zonas de Operación**: Listado de barrios o localidades.

## 2. Contenido Multimedia (El "Cómo se ve")
- [ ] **Foto de Perfil**: Una foto profesional y cercana para la sección "Sobre Mí".
- [ ] **Fotos de Acción**: Mínimo 10-15 fotos reales (clases, eventos, gente bailando).
- [ ] **Fotos de Servicios**: Imágenes que representen cada servicio específico.
- [ ] **Logo**: Si existe, en formato PNG transparente o SVG.

## 3. Datos de Contacto (El "Hablemos")
- [ ] **Número de WhatsApp**: Con código de país (+54).
- [ ] **Links de Redes**: Instagram, TikTok, YouTube, Facebook.
- [ ] **Email de Recepción**: Para configurar formularios si fuera necesario.

## 4. Referencias Estéticas (El "Look")
- [ ] **Estilo Visual**: (Ej: Dark Mode, Minimalista, Futuroista, Energético).
- [ ] **Paleta de Colores**: Un color primario y uno de acento (Ej: Negro y Naranja).

## 5. Acceso Técnico (El "Dónde")
- [ ] **Cuenta de Netlify**: Para el despliegue gratuito.
- [ ] **Cuenta de Formspree**: Para la gestión de formularios (opcional).
