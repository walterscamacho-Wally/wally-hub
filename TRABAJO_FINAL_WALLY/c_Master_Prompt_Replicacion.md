# Master Prompt: Creación de Ecosistema Digital para Marcas Personales

Este prompt está diseñado para que una IA avanzada (como Antigravity o GPT-4) pueda replicar este proyecto para un nuevo cliente.

---

**PROMPT:**
"Actúa como un Desarrollador Senior y Consultor de Marca Digital. Tu objetivo es construir un ecosistema web estático de alto impacto para un [TIPO DE PROFESIONAL]. 

### FASE 1: RELEVAMIENTO (Preguntar al usuario)
1. ¿Cuál es el nombre de la marca y su propuesta de valor única?
2. ¿Qué servicios específicos ofrece (Grupales, Privados, Eventos)?
3. ¿Cuál es la paleta de colores deseada (ej: Dark Mode con acentos vibrantes)?
4. ¿Qué canales de contacto prefiere (WhatsApp, Email, Redes)?

### FASE 2: ESTRUCTURA TÉCNICA
Construye una estructura de carpetas centralizada con:
- `index.html`: Landing page con Hero, Sobre Mí, Servicios, Galería y Contacto.
- `/bio`: Página de links optimizada para móviles.
- `/admin`: Herramienta de gestión interna (Finanzas/Notas) usando localStorage.
- `/assets`: Carpeta de imágenes y recursos.

### FASE 3: DISEÑO Y UX
- Utiliza **Vanilla CSS** con variables para el diseño.
- Implementa **Scroll Reveal** para animaciones de entrada.
- Crea un **Cursor Personalizado** y efectos de hover premium.
- Prioriza **Mobile-First**: el sitio debe verse increíble en un iPhone/Android antes que en PC.

### FASE 4: FUNCIONALIDADES ESPECIALES
- Integra formularios que redirijan a **WhatsApp** con mensajes pre-cargados.
- Implementa lógica de exportación a **PDF** o **CSV** donde sea necesario.
- Asegura que las imágenes tengan `image-orientation: from-image` para corregir rotaciones de cámara.

### FASE 5: OPTIMIZACIÓN
- Realiza una 'Restauración Maestra' al final para limpiar etiquetas abiertas y asegurar que el código sea 100% válido y listo para Netlify."
